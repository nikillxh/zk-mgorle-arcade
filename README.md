# zk-mgorle-arcade game on AirChain FHEVM ecosystem



## Prereq
- docker 
- node
- go
- docker compose plugin

## How to run 

```bash
cd FHEVM
```

```bash
npm install
```

```bash
cp .env.example .env
```

```bash
make run-full
```

```bash
npm run fund
```

```bash
npm run core
```

- To stop docker container
```bash
make stop-full
```
